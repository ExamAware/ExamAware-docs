> **在 Windows 7 中使用 ExamShowboard 是不现实的。** 受限制于框架问题，您无法在 Windows 7 中运行 ExamShowboard 。

# 安装软件

下载完成后，运行安装程序，等待进度条走完后双击自动创建的 `ExamShowboard` 桌面快捷方式即可进入应用
