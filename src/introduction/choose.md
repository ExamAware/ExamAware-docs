## 检查系统版本

> [!note]
> 在使用桌面端或移动端时，请确保您的设备系统满足以下要求!

- 桌面端
  - Windows 系统
    - Windows 10 及以上版本，x86_64 架构   
  - Linux 系统
    - 待定
  - macOS 系统
    - macOS 13 及以上版本
- 移动端
  - 安卓系统
    - 安卓 7 及以上版本
  - iOS 系统
    - iOS 13 及以上版本
